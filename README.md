# Projeto-IDF
Projeto acadêmico desenvolvido no primeiro ano do ensino médio ao longo de aproximadamente 4 meses, com uma equipe composta por 4 membros, me incluindo. A iniciativa visava abordar a problemática da indisponibilidade de medicamentos nas farmácias, culminando na concepção de um website dotado de diversas funcionalidades para solucionar essa questão.
